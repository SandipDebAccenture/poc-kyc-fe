# poc-kyc-fe (POC branch)

Lightweight proof-of-concept React + TypeScript frontend for KYC registration.

**Prerequisites**

- **Node.js** v16 or newer
- **npm**, **yarn**, or **pnpm**

**Install**

```bash
npm install
```

**Run (development)**

```bash
npm run dev
```

**Build & Preview**

```bash
npm run build
npm run preview
```

**Notes**

- This repository is a proof-of-concept (POC) branch.
- Main form component: [src/components/UserKycRegistrationForm/UserKycRegistrationForm.tsx](src/components/UserKycRegistrationForm/UserKycRegistrationForm.tsx)
- App entry: [src/App.tsx](src/App.tsx)
- If scripts differ, check [package.json](package.json) for exact commands.

Feel free to open an issue or update the README with environment-specific instructions.
